Using this reference in Cursor: Use token names as variable names in your CSS/Tailwind config. For semantic tokens, switch the value based on the active theme (e.g. color/surface/page-bg → #FAF8F2 in light mode, #0C0C0C in dark). Spacing tokens are multiples of 4px. Border radius tokens are scoped to corner radius only. Typography variables cover size and line-height; all use Montserrat with weights 400 / 500 / 600 / 700.
